# 贪吃蛇小游戏

一个开箱即用的纯前端贪吃蛇小游戏。

## 运行方式

### 方式 1：直接打开

直接在浏览器中打开 `index.html`。

### 方式 2：使用内置 Node 静态服务（推荐）

```bash
npm start
```

启动后访问 <http://localhost:8000>。

> 如果你之前使用 `python3 -m http.server` 出现 `not found`，可以直接用上面的 `npm start`，不依赖 Python。

## 操作说明

- 方向键 / `WASD`：控制移动
- 空格：暂停 / 继续
- 重新开始按钮：重置游戏

## 游戏规则

- 吃到食物会加分并增长身体
- 分数越高速度越快
- 撞墙或撞到自己则游戏结束
- 最高分会记录在浏览器本地存储中
